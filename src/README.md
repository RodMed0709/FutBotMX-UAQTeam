# src (SOURCE)

Directorio principal del proyecto destinado a albergar el código fuente limpio, optimizado y estructurado por módulos.

Flujo de desarrollo:

- Modularización: Toda la lógica de negocio, procesamiento y utilidades debe estar desacoplada en módulos independientes.
- Transición desde Notebooks: Todo prototipo o funcionalidad validada en el directorio de notebooks/ que sea útil para el proyecto debe ser refactorizado, estructurado y trasladado a este directorio bajo estándares de código limpio (Clean Code).
