# Models

Directorio destinado a almacenar los artefactos y archivos generados durante el ciclo de vida de los modelos.

Aquí se deben organizar:

- Checkpoints: Estados guardados durante el entrenamiento.
- Pesos (Weights): Parámetros optimizados del modelo.
- Modelos intermedios y finales: Versiones listas para evaluación, inferencia o despliegue.

Debido al gran tamaño de estos archivos, su contenido está excluido del repositorio de Git. Este directorio se conserva en el control de versiones únicamente para preservar la estructura y arquitectura del proyecto.
