# Notebooks

Directorio destinado a almacenar los notebooks de desarrollo y experimentación utilizados por los integrantes del equipo.

Estos archivos funcionan como un entorno de pruebas (playground) para:

- Explorar y definir la lógica del proyecto.
- Desarrollar prototipos e integraciones (ej. SAM 3).
- Realizar pruebas visuales y validación de conceptos.

Consideraciones:

- Trabajo individual: Cada integrante debe mantener sus notebooks identificados de manera individual para evitar conflictos de fusión (merge conflicts) en Git.

- Integración con Google Colab: Se permite la conexión de Google Colab con este repositorio para guardar y actualizar directamente los avances en este directorio.
